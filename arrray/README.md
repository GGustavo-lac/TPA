# Exemplos de Uso dos Códigos da Pasta `arrray`

Este diretório contém três programas em Java para exercício com arrays. Abaixo estão exemplos de execução (prints dos outputs) para facilitar testes e compreensão do funcionamento de cada arquivo.

---

## 1. Numerosimpares.java

Recebe 100 números do usuário e imprime apenas os números ímpares.

**Exemplo:**
```
Digite 100 números:
Número 1: 5
Número 2: 8
Número 3: 13
...              <!-- sequência 100 números -->
Número 100: 99

Números ímpares digitados:
5
13
...
99
```

---

## 2. Numerospares.java

Recebe 100 números do usuário e imprime apenas os números pares.

**Exemplo:**
```
Digite 100 números:
Número 1: 7
Número 2: 4
Número 3: 12
...
Número 100: 100

Números pares digitados:
4
12
...
100
```

---

## 3. nomesordem.java

Recebe nomes do usuário e imprime em ordem crescente (alfabética) e decrescente.

**Exemplo:**
```
Quantos nomes deseja digitar? 3
Digite o nome 1: Carlos
Digite o nome 2: Ana
Digite o nome 3: Beatriz

Nomes em ordem crescente:
Ana
Beatriz
Carlos

Nomes em ordem decrescente:
Carlos
Beatriz
Ana
```
